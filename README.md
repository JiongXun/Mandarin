### 不-  (죰 어려운)

동사 앞에서 不

	너 밥 먹어요?? 你 吃 飯 嗎?
	* 네,전 먹다. 嗯,我 吃.
	* 아니요,전 안 먹다. 不, 我 不 吃.
	* 아니요,전 못 먹다. 不, 我 不能(ㄋㄥˊ) 吃.
		能 는 영어의 can 느낌이에요.
			I can eat. 전 먹고 수 있다.  我能吃
			I can not eat.전 먹고 수 없다.  我不能吃

	A가: 이건 맞아요?? (這是對的嗎?)
	B가: 아니요. (不對)
